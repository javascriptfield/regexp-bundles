## Usage

### Install

```sh
# npm
npm i  regexp-bundle
# yarn 
yarn add regexp-bundle
# pnpm
pnpm add regexp-bundle
```

#### Full Import

```js
import * as MyRegExp from 'regexp-bundle'
const IDCardRegExp = MyRegExp.IDCardNumber
```

#### On-demand Import

```js
import { IDCardNumber } from 'regexp-bundle'
const IDCardRegExp = IDCardNumber
```
